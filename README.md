PARA EL FRONTEND:
ejecutar en la consola:
pnpm i
pnpm run server
(en otra consola, mismo vs) pnpm run dev

----------------------------------------------------------------------------

PARA EL BACKEND:
ejecutar en la consola (Windows PowerShell):
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python -m fastapi dev main.py
